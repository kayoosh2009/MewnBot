Идея для проекта:
MewnBot - бот по озвучке чата твича и добавления виджетов в обс

Бот работает по этапно и на несколько << фронтов >> одновременно.
1. Сканирования чатов - сканирует чат стрима YouTube и Twitch и сохраняет чаты ввиде [ T ] nickname: message и сохраняет его в temp_chat.jsonl и history_chat.jsonl
    [ T ] - платформа T это твич а Y это ютуб
    nickname - ник пользователя
    message - само сообщение
2. Озвучка чата - озвучивает при помощи TTS чат получая первую строчку из temp_chat.jsonl, озвучивает его и удаляет эту строчку и потом через пару секунд озвучивает следующую строчку
3. Виджет для OBS - есть 2 виджета которые запускаются через локал хост которые просто добавляются в OBS
    Виджет чата - просто виджет чата который показывает чат влево показывая сообщения которые присылаются в чат, так же зритель будет показываться с тенью цвета платформы откуда он пишет сообщение.
        [ T ] - фиолетовая тень у ника
        [ Y ] - красная тень у ника
    Виджет топ чата - данный виджет показывает топ 5 чатеров ( люди присылающие сообщения ), статистика находится в state.json и показыааются топ 5 чатеров за все время, первое место подсвечивается золотым цветом, второе серебряным а третий медным - а так же 4 и 5 место серыми ( при увеличения счетчика то ник будет чуть чуть увеличиваться ) 

Файлы которые будут в проекте
1. MewnBot.exe - готовый файл для реализа ( Самый конец и я его сам сделаю )
2. main.py - Главный код на котором все работает
3. history_chat.jsonl - История всего чата за все время
4. temp_chat.jsonl - История чата которая удаляется после озвучки
5. stats.json - Статистика каждого пользователя по нику
