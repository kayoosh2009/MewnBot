# 🐾 MewnBot

**MewnBot** — это мощный локальный инструмент для Twitch-стримеров, созданный для того, чтобы полностью заменить второй монитор с чатом и максимально вовлечь аудиторию в процесс стрима.

---

## 🚀 Основные возможности

MewnBot делает ваш стрим живым и интерактивным без использования сторонних тяжелых сервисов:

1.  **💬 Интерактивный Чат** — стильный виджет чата для OBS (работает через localhost).
2.  **🎉 GIF-реакции** — мгновенное появление гифок на экране при каждом новом сообщении.
3.  **🏆 ТОП-5 Чатеров** — динамический лидерборд самых активных зрителей в реальном времени.
4.  **🎙️ Умная Озвучка (TTS)** — бот читает сообщения из чата голосом прямо в эфире.
5.  **🤖 Персональный Бот** — ваш собственный бот в чате Twitch для автоматических уведомлений.

---

## 🛠 Как это работает?

MewnBot работает локально на вашем компьютере, обеспечивая минимальную задержку и полную приватность данных. Все виджеты подключаются в OBS как "Браузер" через локальные ссылки.

---

## 📦 Быстрый старт

1.  Скачайте актуальный `.exe` файл из раздела [Releases](ссылка_на_релиз).
2.  Запустите программу на своем ПК.
3.  Введите название вашего Twitch-канала.
4.  Скопируйте сгенерированные ссылки и добавьте их в **OBS** как источник "Браузер".
5.  Наслаждайтесь новым уровнем взаимодействия с чатом!
