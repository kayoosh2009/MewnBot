# MewnBot
MewnBot это специальный бот разработаный для стримеров как замена второму монитору с чатом твича.

Что умеет бот?
1. Добавляет ссылку для отображения чата на стриме ( localhost ссылка )
2. Добавляет гифку при написания коментария ( localhost ссылка )
3. Добавляет топ 5 чатеров ( localhost ссылка )
4. Озвучка чата в прямом эфире
5. Личный бот в чате для текстовых уведомлений 

Как включить?
Скачайте exe файл и запустите его, укажите имя вашего канала и наслаждайтесь вайбами стримерства!
